# CSV 报表导出功能设计文档

## 项目背景

当前广告 BI 系统缺少数据导出功能，用户无法将数据分析结果保存到本地进行进一步处理或归档。本设计旨在实现一个简单、实用的 CSV 报表导出功能。

## 需求分析

### 用户需求
- 支持自定义时间范围选择
- 导出 CSV 格式报表
- 包含核心指标概览和每日趋势数据
- 简单易用的交互体验

### 功能范围
- 在数据看板页面添加导出按钮
- 支持选择起始日期和结束日期
- 导出 CSV 文件包含概览数据和每日趋势

## 技术方案

### 方案选择
采用**纯前端导出方案**，理由如下：
1. 当前项目数据量较小，纯前端导出完全够用
2. 实现简单快速，能尽快看到效果
3. 符合现有技术栈（Vue 3），无需额外依赖
4. 用户体验好，点击即可下载，无需等待

### 技术实现

#### 1. 组件结构

```
Dashboard.vue
├── 导出按钮（页面右上角）
├── 时间选择对话框（el-dialog）
│   ├── 日期范围选择器（el-date-picker）
│   ├── 确认按钮
│   └── 取消按钮
└── CSV 生成工具函数（内部方法）
```

#### 2. 核心功能

**数据收集**：
- 概览数据：从 `overview` 接口获取
- 每日趋势数据：从 `daily-trend` 接口获取

**CSV 生成**：
- 使用 JavaScript 将数据转换为 CSV 格式
- 处理中文编码问题（UTF-8 with BOM）
- 创建 Blob 对象并触发下载

#### 3. CSV 文件格式

**文件命名**：
```
report_YYYY-MM-DD_to_YYYY-MM-DD.csv
```

**文件内容**：
```
广告 BI 系统数据报表
时间范围: 2024-01-01 至 2024-01-31
导出时间: 2024-01-31 15:30:00

指标,数值
总曝光量,12586342
总点击量,892345
总转化数,45678
总收入,2895678.50
平均点击率,7.09%
平均转化率,5.12%

日期,曝光量,点击量,转化数,收入
2024-01-01,1856234,125678,6789,423456.78
2024-01-02,2134567,145678,7890,487654.32
...
```

#### 4. 交互流程

```
用户点击"导出报表"按钮
    ↓
弹出时间选择对话框（默认选择最近7天）
    ↓
用户选择起始日期和结束日期
    ↓
点击"导出"按钮
    ↓
前端生成 CSV 文件
    ↓
自动下载 CSV 文件到本地
    ↓
显示"导出成功"提示
```

## 实现步骤

### 步骤 1：修改 Dashboard.vue

1. 添加导出按钮到页面头部
2. 添加时间选择对话框组件
3. 实现 CSV 生成和下载逻辑

### 步骤 2：测试验证

1. 测试默认日期范围导出
2. 测试自定义日期范围导出
3. 测试边界情况（无数据、日期错误等）

## 边界情况处理

1. **未选择日期范围**：默认导出最近7天数据
2. **起始日期晚于结束日期**：显示错误提示，不执行导出
3. **没有数据**：显示"暂无数据可导出"提示
4. **网络请求失败**：显示错误提示

## 成功标准

1. 用户可以点击导出按钮打开时间选择对话框
2. 用户可以选择自定义日期范围
3. 用户可以成功导出 CSV 文件
4. CSV 文件包含完整的概览数据和每日趋势数据
5. 导出失败时有明确的错误提示
